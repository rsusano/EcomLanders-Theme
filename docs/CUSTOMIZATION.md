# Customization Guide

All customization is done via the Shopify Theme Editor — no code changes required.

## Product Info Advanced

1. Add the section to your product template
2. Use the sidebar to configure blocks: bundle cards, trust signals, rating, benefits, media carousel
3. Adjust colors, fonts, spacing in the section settings

## Product Announcement Bar

1. Add the section to your preferred template (header, main, or custom)
2. Add announcement blocks for each message
3. Toggle scrolling mode and separator style as needed
